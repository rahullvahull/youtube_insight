# Contributing to YouTube Wrapped

Thank you for your interest in contributing! 🎉

## 🔒 Repository Policy

This repository is **view and download only** for the general public. Direct edits to the main branch are not permitted.

## 📝 How to Contribute

### For Bug Reports or Feature Requests:
1. Go to the [Issues](https://github.com/rahullvahull/youtube_insight/issues) tab
2. Click **"New Issue"**
3. Choose the appropriate template:
   - 🐛 **Bug Report** - For reporting bugs
   - 💡 **Feature Request** - For suggesting new features
4. Fill out the template with as much detail as possible
5. Submit and wait for a response

### For Code Contributions:
If you'd like to contribute code:

1. **Fork** this repository to your own GitHub account
2. **Clone** your fork locally
3. Create a new **branch** for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. Make your changes and **commit** them:
   ```bash
   git commit -m "Add: your feature description"
   ```
5. **Push** to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
6. Open a **Pull Request** from your fork to this repository
7. Wait for review and approval

## ✅ Pull Request Guidelines

- Provide a clear description of what your PR does
- Reference any related issues (e.g., "Fixes #123")
- Ensure your code works and doesn't break existing functionality
- Follow the existing code style and structure

## 🚫 What Will Be Rejected

- Direct pushes to `main` branch
- PRs without proper description
- Changes that compromise user privacy
- Features that require server-side processing

## 📧 Questions?

If you have questions, please open an issue with the "question" label.
