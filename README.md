# YouTube Wrapped (Beta)

A privacy-focused web application that analyzes your YouTube watch history from Google Takeout and generates a beautiful "Spotify Wrapped" style dashboard.

## 🎯 Features

- **📊 Comprehensive Analytics**
  - Total videos watched and unique creators
  - Estimated watch time with precise hour/minute breakdown
  - GitHub-style contribution heatmap for the last 365 days
  
- **📈 Visual Insights**
  - Monthly activity trends
  - Weekly viewing patterns
  - 24-hour activity distribution
  
- **🎬 Detailed Breakdowns**
  - Top creators leaderboard with watch time
  - Top 100 videos with thumbnails
  - Peak viewing times for each video

## 🔒 Privacy First

All data processing happens **locally in your browser**. Your YouTube history never leaves your device - no servers, no tracking, no data collection.

## 🚀 How to Use

1. **Get Your Data**: Request your YouTube data from [Google Takeout](https://takeout.google.com/)
   - Select only "YouTube and YouTube Music"
   - Choose "watch-history.json" format
   - Download the ZIP file

2. **Upload & Analyze**: 
   - Visit the [YouTube Wrapped Dashboard](https://YOUR_USERNAME.github.io/youtube_wrapped/)
   - Drag and drop your ZIP file
   - View your personalized insights instantly!

## 🛠️ Tech Stack

- **HTML5** - Single-file application
- **Tailwind CSS** - Modern, responsive styling
- **Chart.js** - Beautiful data visualizations
- **JSZip** - Client-side ZIP file processing
- **Vanilla JavaScript** - No frameworks, pure performance
